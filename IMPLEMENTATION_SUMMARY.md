## 二、技术实现细节

### 2.1 前端架构

#### 核心组件
- [MindMapView.vue](src/views/MindMapView.vue) - 思维导图编辑器
- [ChatPanel.vue](src/components/ChatPanel.vue) - AI 对话面板
- [ExportDialog.vue](src/components/ExportDialog.vue) - 导出对话框
- [ConversationList.vue](src/components/ConversationList.vue) - 会话列表

#### 状态管理
- useDraftStore - 草稿管理
- useConversationStore - 会话管理
- useAIChat - AI 对话逻辑

#### API 层
- mindmapApi - 思维导图 API
- conversationApi - 会话 API
- messageApi - 消息 API
- exportApi - 导出 API（新增）

### 2.2 后端支持

#### Rust 模块
- db.rs - SQLite 数据库操作
- commands.rs - Tauri 命令处理
- export.rs - 文档导出逻辑
- sandbox.rs - OpenClaw 沙箱管理

#### 数据库表
- mindmaps - 思维导图表
- conversations - 会话表
- messages - 消息表
- skills - 技能模板表
