## 🎯 三大高优先级任务

### 🔴 1. OpenClaw 沙箱集成 (最优先)
**现状**: AI 对话为模拟响应，未连接真实 AI  
**价值**: 核心价值无法体现

**关键里程碑**:
- [ ] OpenClaw HTTP 客户端实现
- [ ] SSE 流式响应处理
- [ ] 错误处理和重试机制

**预计时间**: 1 周

---

### 🔴 2. 思维导图 ↔ AI 对话双向关联 (核心价值)
**现状**: 两个模块独立工作  
**价值**: "选中节点 → AI 分析 → 结果导入"完整工作流

**交互流程**:
```
选中思维导图节点 → 点击"AI 分析" 
    ↓
自动携带上下文发起对话
    ↓
AI 生成建议
    ↓
一键导入为子节点
```

**关键里程碑**:
- [ ] 思维导图 → Markdown 转换
- [ ] AI 回复 → 节点结构解析
- [ ] 导入确认和撤销支持

**预计时间**: 1 周

---

### 🔴 3. Skill 系统完善 (差异化竞争力)
**现状**: 基础 CRUD 已完成，变量系统未实现  
**价值**: 差异化 Prompt 模板，提高 AI 输出质量

**预设模板**:

| 模板 | 使用场景 |
|------|---------|
| PRD 生成器 | 快速生成产品需求文档 |
| 竞品分析 | 生成竞品对比报告 |
| 用户故事地图 | 生成用户故事地图 |
| 需求评审 | 生成评审检查清单 |
| 会议纪要 | 生成结构化会议纪要 |
| 周报生成 | 生成格式化周报 |

**关键里程碑**:
- [ ] SkillVariableForm 组件
- [ ] 变量验证逻辑
- [ ] 6 个预设模板
- [ ] Skill 调用界面

**预计时间**: 1.5 周

## ⚠️ 核心风险

1. **OpenClaw 可用性**
   - 影响: 中 | 概率: 低
   - 建议: 准备备选方案（直接 API）

2. **Token 限制**
   - 影响: 低 | 概率: 高
   - 建议: 智能截断和分片

3. **解析准确性**
   - 影响: 中 | 概率: 中
   - 建议: 提供人工确认环节
