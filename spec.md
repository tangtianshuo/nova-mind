# Nova Mind 功能规划文档

> 版本: v1.0.0  
> 更新日期: 2026-03-26

## 一、项目概述

### 1.1 项目定位
Nova Mind 是一个 AI 驱动的产品管理思维导图桌面应用，帮助 PM 团队更高效地管理和可视化产品需求。

### 1.2 核心技术栈
- **前端**: Vue 3 + TypeScript + Tailwind CSS
- **后端**: Tauri 2.0 (Rust)
- **数据库**: SQLite
- **AI 引擎**: OpenClaw 沙箱

### 1.3 核心模块
1. 思维导图编辑
2. AI 智能对话
3. 技能(Skill)管理
4. 会话管理
5. 文档导出

---

## 二、核心功能详细规划

### 2.1 思维导图编辑 [P0]

#### 2.1.1 节点管理
- **创建节点**: 支持在任意节点下创建子节点
- **编辑节点**: 双击节点进入编辑模式，支持文本输入
- **删除节点**: 支持删除单个节点及其所有子节点
- **节点样式**: 支持自定义节点颜色、字体大小

#### 2.1.2 节点操作
- **拖拽移动**: 自由拖拽节点调整位置
- **展开/折叠**: 支持子节点的展开和折叠
- **批量选择**: 支持多选节点进行批量操作
- **撤销/重做**: 支持操作历史的撤销和重做

#### 2.1.3 画布操作
- **缩放**: 支持鼠标滚轮缩放画布
- **平移**: 支持鼠标拖拽平移画布
- **自适应布局**: 一键自动布局
- **网格吸附**: 可选的网格对齐功能

#### 2.1.4 数据持久化
- **自动保存**: 定时自动保存（默认每 30 秒）
- **手动保存**: 支持 Ctrl+S 快捷键保存
- **版本历史**: 保留最近 10 个版本的历史记录

#### 2.1.5 导出功能
- **Markdown 导出**: 导出为标准 Markdown 格式
- **Word 导出**: 导出为 .docx 格式
- **图片导出**: 导出为 PNG/SVG 图片

#### 2.1.6 交互细节
- 节点悬停: 显示操作按钮（添加、删除、编辑）
- 节点点击: 选中节点，高亮显示
- 右键菜单: 上下文菜单提供更多操作
- 键盘快捷键: Delete 删除、Enter 编辑、Tab 添加子节点

---

### 2.2 AI 智能对话 [P0]

#### 2.2.1 对话管理
- **创建会话**: 支持创建新的对话会话
- **会话列表**: 显示所有历史会话，支持搜索
- **会话切换**: 快速切换不同会话
- **会话重命名**: 支持自定义会话标题
- **删除会话**: 支持删除不需要的会话

#### 2.2.2 消息交互
- **发送消息**: 支持文本输入发送消息
- **流式响应**: 支持 SSE 流式输出 AI 响应
- **Markdown 渲染**: AI 响应支持 Markdown 格式
- **代码高亮**: 代码块语法高亮显示
- **复制代码**: 一键复制代码块内容

#### 2.2.3 上下文关联
- **思维导图关联**: AI 对话可与当前思维导图关联
- **上下文传递**: 自动将关联的思维导图内容作为上下文
- **Skill 调用**: 支持选择预设的 Skill 模板

#### 2.2.4 AI 能力集成
- **OpenClaw 集成**: 通过 Rust 后端调用 OpenClaw
- **SSE 流式**: 实现 Server-Sent Events 流式响应
- **错误处理**: 优雅处理网络错误、超时等问题
- **加载状态**: 显示 AI 处理中的加载状态

#### 2.2.5 交互细节
- Enter 发送、Shift+Enter 换行
- 消息发送动画效果
- AI 响应打字机效果
- 空消息拦截提示

---

### 2.3 技能(Skill)管理 [P1]

#### 2.3.1 Skill 基础功能
- **创建 Skill**: 支持创建新的 Prompt 模板
- **编辑 Skill**: 修改现有 Skill 内容
- **删除 Skill**: 删除不需要的 Skill
- **分类管理**: 支持为 Skill 添加分类标签

#### 2.3.2 Skill 模板
- **变量支持**: Skill 支持定义输入变量
- **变量类型**: 支持文本、选择框、数字、日期等类型
- **变量验证**: 支持必填、最小/最大长度等验证规则
- **默认值**: 支持设置变量默认值

#### 2.3.3 预设 Skill
- **PRD 生成**: 产品需求文档生成
- **竞品分析**: 竞品对比分析模板
- **用户故事**: 用户故事地图生成
- **需求评审**: 需求评审检查清单

#### 2.3.4 Skill 使用
- **快速调用**: 在 AI 对话中快速选择 Skill
- **变量填充**: 调用时弹出变量填充表单
- **历史记录**: 记录 Skill 的使用历史

---

### 2.4 会话管理 [P1]

#### 2.4.1 会话存储
- **SQLite 存储**: 所有会话数据持久化存储
- **消息历史**: 保留完整的对话历史
- **关联思维导图**: 支持会话与思维导图关联

#### 2.4.2 会话操作
- **新建会话**: 创建空白会话
- **重命名会话**: 自定义会话标题
- **删除会话**: 删除会话及其所有消息
- **清空消息**: 保留会话但清空消息

#### 2.4.3 会话列表
- **按时间排序**: 默认按最近更新时间排序
- **搜索功能**: 支持按标题搜索会话
- **分类筛选**: 支持按关联的思维导图筛选

---

### 2.5 主题切换 [P2]

#### 2.5.1 主题模式
- **浅色模式**: 白色背景的浅色主题
- **深色模式**: 深色背景的暗色主题
- **自动模式**: 跟随系统设置自动切换

#### 2.5.2 主题应用
- **全局应用**: 所有页面和组件支持主题切换
- **平滑过渡**: 主题切换使用平滑过渡动画
- **记忆恢复**: 记住用户上次选择的主题

---

### 2.6 草稿自动保存 [P2]

#### 2.6.1 草稿机制
- **自动保存**: 输入内容自动保存为草稿
- **草稿恢复**: 刷新页面后自动恢复未发送的内容
- **草稿清理**: 发送成功后自动清理相关草稿

#### 2.6.2 保存策略
- **定时保存**: 每 5 秒自动保存草稿
- **即时保存**: 内容变化后立即保存
- **存储位置**: 本地 LocalStorage

---

### 2.7 快捷键支持 [P2]

#### 2.7.1 全局快捷键
- `Ctrl+N`: 新建思维导图
- `Ctrl+S`: 保存当前内容
- `Ctrl+E`: 导出文档
- `Ctrl+,`: 打开设置
- `Ctrl+K`: 打开命令面板

#### 2.7.2 思维导图快捷键
- `Delete`: 删除选中节点
- `Enter`: 编辑选中节点
- `Tab`: 添加子节点
- `Escape`: 取消编辑
- `Ctrl+Z`: 撤销
- `Ctrl+Y`: 重做

#### 2.7.3 对话快捷键
- `Enter`: 发送消息
- `Shift+Enter`: 换行
- `Ctrl+L`: 清空对话
- `Ctrl+F`: 搜索会话

## 三、技术实现方案

### 3.1 前端架构

#### 3.1.1 状态管理 (Pinia)
```
stores/
├── themeStore.ts      # 主题状态管理
├── skillStore.ts      # Skill 数据管理
├── conversationStore.ts # 会话数据管理
├── draftStore.ts      # 草稿数据管理
├── settingsStore.ts   # 设置数据管理
└── shortcutStore.ts   # 快捷键管理
```

#### 3.1.2 组件结构
```
components/
├── AppSidebar.vue     # 侧边导航栏
├── AppHeader.vue      # 顶部标题栏
├── CustomTitleBar.vue # 自定义标题栏
├── ChatPanel.vue     # AI 对话面板
├── ConversationList.vue # 会话列表
├── MarkdownRenderer.vue # Markdown 渲染器
├── SkillCard.vue     # Skill 卡片组件
├── SkillEditor.vue   # Skill 编辑器
├── VirtualMessageList.vue # 虚拟滚动消息列表
└── ...
```

#### 3.1.3 视图结构
```
views/
├── HomeView.vue       # 首页
├── MindMapView.vue    # 思维导图编辑
├── ChatView.vue       # AI 对话
├── SkillListView.vue  # Skill 管理
└── SettingsView.vue   # 设置页面
```

### 3.2 后端架构 (Rust)

#### 3.2.1 模块划分
```
src-tauri/src/
├── main.rs           # 应用入口
├── db.rs             # 数据库操作
├── commands.rs       # Tauri 命令
├── sandbox.rs        # OpenClaw 沙箱管理
├── export.rs         # 文档导出
├── health.rs         # 健康检查
└── security.rs       # 安全相关
```

#### 3.2.2 数据库表结构
- `mindmaps`: 思维导图表
- `skills`: 技能模板表
- `conversations`: 会话表
- `messages`: 消息表

### 3.3 API 设计

#### 3.3.1 思维导图 API
- `get_mindmaps`: 获取所有思维导图
- `create_mindmap`: 创建思维导图
- `get_mindmap`: 获取单个思维导图
- `update_mindmap`: 更新思维导图
- `delete_mindmap`: 删除思维导图

#### 3.3.2 AI 对话 API
- `get_conversations`: 获取所有会话
- `create_conversation`: 创建会话
- `get_messages`: 获取会话消息
- `create_message`: 发送消息
- `stream_response`: SSE 流式响应

#### 3.3.3 Skill API
- `get_skills`: 获取所有 Skill
- `create_skill`: 创建 Skill
- `update_skill`: 更新 Skill
- `delete_skill`: 删除 Skill

### 3.4 性能优化

#### 3.4.1 前端优化
- **虚拟滚动**: 长消息列表使用虚拟滚动
- **懒加载**: 路由组件懒加载
- **防抖节流**: 输入事件防抖处理
- **缓存策略**: 合理使用计算属性缓存

#### 3.4.2 后端优化
- **数据库索引**: 为常用查询添加索引
- **连接池**: 使用连接池管理数据库连接
- **异步操作**: 使用 async/await 处理 IO 操作
